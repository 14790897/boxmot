## 核心功能

✅ **三种输入方式**
- 上传视频
- 输入文件夹路径
- 批量文件配置

✅ **配置管理**
- Web界面配置保存目录
- 配置文件持久化

✅ **文件数量限制**
- 快速测试：处理前N个文件
- 完整处理：处理所有文件

✅ **批量处理**
- 从配置文件读取多对目录
- 支持注释和中文路径
