# 配置说明

> **完整使用指南**: 请查看 [USER_GUIDE.md](USER_GUIDE.md)

## 配置文件

**位置**: `config.json` (项目根目录)

```json
{
  "yolo_save_directories": {
    "y_track_project": "runs/track",
    "y_track_name": "exp",
    "x_detect_project": "runs_x_me/detect",
    "x_detect_name": "exp",
    "video_output": "processed_video_gradio"
  },
  "processing_options": {
    "max_files_per_folder": null
  }
}
```

## 配置项

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `y_track_project` | `runs/track` | Y轴跟踪结果保存目录 |
| `y_track_name` | `exp` | Y轴实验名称（自动递增: exp, exp2...） |
| `x_detect_project` | `runs_x_me/detect` | X轴检测结果保存目录 |
| `x_detect_name` | `exp` | X轴实验名称（自动递增: exp, exp2...） |
| `video_output` | `processed_video_gradio` | 视频输出目录 |
| `max_files_per_folder` | `null` | 每个文件夹最大处理文件数（null=全部） |

## 修改方式

### 方式1: Web界面（推荐）

1. 启动: `gradio app.py`
2. 展开 "YOLO Save Directory Configuration"
3. 修改路径
4. 点击 "Save Configuration"

### 方式2: 直接编辑

1. 打开 `config.json`
2. 修改对应的值
3. 保存文件
4. 重启应用

## 注意事项

- 所有路径相对于项目根目录
- 目录不存在会自动创建
- 配置修改后立即生效
- 使用 "clear folder" 按钮清理输出目录
